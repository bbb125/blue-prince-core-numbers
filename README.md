## Blue Prince Late Game Puzzle solve
Brute force solution for the late game puzzle in Blue Prince.
Displaying possible combinations of letters in each slot of each word.
```plaintext
SL,TJ,IPC,LO,LACP,
WAL,AFJ,TOS,EJX,R,
TKC,IN,N,TCK,SLP,
B,LKC,APBE,NJAV,KFH,
BXJ,ON,OAY,KMX,SPN,
```

It seems the puzzle was designed for brute forcing, as the first letters
form perfect phrase:
```plaintext
STILL
WATER
TINTS
BLANK
BOOKS
```
